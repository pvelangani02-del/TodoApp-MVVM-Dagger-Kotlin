# TodoApp MVVM Dagger Kotlin

## Overview
A simple Todo application built using the MVVM architecture pattern, Dagger for dependency injection, and Kotlin programming language.

## Features
- Modern UI design
- Add, edit, and delete todo items
- Mark tasks as complete

## Technologies Used
- **Kotlin**: A modern programming language that is fully interoperable with Java.
- **Android Jetpack**: Set of libraries for Android development.
- **Dagger**: A dependency injection framework for Java and Android.

## Architecture
This project follows the MVVM design pattern:
- **Model**: Represents the data and business logic.
- **View**: The UI components that display the data.
- **ViewModel**: Acts as a bridge between the Model and View.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/pvelangani02-del/TodoApp-MVVM-Dagger-Kotlin.git
   ```
2. Open the project in Android Studio.
3. Build and run the project.

## Usage
- Launch the app from your device or emulator.
- Use the '+' button to add a new todo item.
- Tap on a todo item to edit or delete it.
